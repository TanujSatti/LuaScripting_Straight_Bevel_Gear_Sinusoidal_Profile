# LuaScripting_Straight_Bevel_Gear_Sinusoidal_Profile

Parametric Lua scripts for **IceSl 2.5.x** that generate straight bevel gear pairs with sinusoidal tooth profiles. Designed for precise 3D geometry output suitable for CAD/CAE workflows, prototyping, and 3D printing.

---

## Features

- **Variable gear ratio** вЂ” define any ratio between 0.5Г— and 4Г— via a UI slider
- **Sinusoidal tooth profile** вЂ” smooth, noise-reducing tooth shape with adjustable amplitude, smoothness, and wave frequency
- **Correct bevel cone geometry** вЂ” cone angles are derived directly from tooth counts, ensuring an exact gear ratio at a 90В° shaft angle
- **Miter gear support** вЂ” when N1 = N2 the script automatically produces a 45В°/45В° miter pair
- **Live UI controls** вЂ” all key parameters are exposed as IceSl sliders/toggles; no manual code editing required
- **Shaft bore & hub** вЂ” each gear includes a hub cylinder and a through-bore for the shaft
- **Automatic meshing** вЂ” Gear 2 is phase-shifted and translated so its teeth land in Gear 1's gaps at the correct apex

## How It Works

### Cone Angle Derivation
For a 90В° shaft angle the pitch cone angles are:

```
delta1 = atan(N1 / N2)
delta2 = 90В° - delta1
```

This ensures the two cones share a common apex, which is the fundamental requirement for correct bevel gear meshing.

### Tooth Profile
Each tooth cross-section is generated by modulating the pitch radius with a shaped sine wave:

```lua
r(t) = Rp + A В· sign(sin(NВ·t)) В· |sin(NВ·t)|^sp
```

- `Rp` вЂ” pitch radius  
- `A`  вЂ” amplitude (scales from the large end to the small end along the cone)  
- `sp` вЂ” smoothness exponent (`sine_power` parameter)

The amplitude tapers linearly from the large face to the small face, matching the narrowing cone geometry.

### Body Construction
1. A point cloud is computed at 240 steps around the pitch circle for both the large face (`z = 0`) and the small face (`z = bh`).
2. `sections_extrude` lofts the two contours into a solid tapered body.
3. A hub cylinder is unioned below the large face.
4. A shaft bore is differenced through the entire part.

### Gear 2 Placement
- Rotated 90В° around X to align its axis perpendicular to Gear 1.
- Phase-shifted by half a tooth pitch (`ПЂ / N2`) so teeth interleave correctly.
- Translated so both cones share the same apex point, offset by `mesh_offset` for clearance.
